# Context & Guidelines: DrFit Hackathon App

## Tech Stack
- Expo SDK: 55 (Latest)
- React Native: 0.81.5
- React: 19.1.0
- Engine: New Architecture (Fabric) is ENABLED.

## Constraints & Rules
- Do NOT use deprecated libraries. Always prefer built-in Expo modules (`expo-*`).
- No complex state management (Redux/Zustand) unless strictly necessary. Rely on React Context.
- All code comments must be strictly in English.
- Focus on shipability and clean, pragmatic components.
